Campus Navigation Tool

A smart web-based campus navigation system that helps users locate buildings, departments, and facilities within a campus efficiently. It provides accurate route guidance using distance calculation and pathfinding techniques.

Features:
1. Interactive campus navigation
2. Search for buildings and locations
3. Shortest path calculation
4. Distance calculation using Haversine Formula
5. Responsive and user-friendly UI
6. Dynamic route visualization

Tech Stack:

1. Frontend
- HTML
- CSS
- JavaScript

2. Backend
- Node.js
- Database
- MySQL
